# CLI-Guessing-Game
A simple Python CLI game where players try to guess a randomly generated number.


About The Project

This was one of my first Python projects made while learning loops, conditionals, exception handling, and user interaction.

The game provides feedback based on how far the player's guess is from the actual number. It also tracks:
- number of chances taken
- number of times played

Features

- Random number generation
- High/Low feedback system
- “Way too high/low” logic
- Input validation using try-except
- Replay system
- Guess counter
- Game session counter

Concepts Used

- while loops
- nested if-else
- functions like abs()
- exception handling
- counters and variables
- random module

How To Run

Run the Python file using:
python filename.py
